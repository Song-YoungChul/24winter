LSTM & GRU

============================================
tensorflow keras 기반의 모델 구축

본 모델은 6가지 감정(분노, 기쁨, 불안, 당황, 슬픔, 상처)을 분류해주는 분류 모델임

============================================

사용법

1) 감정에 해당하는 텍스트가 존재한 코퍼스를 준비
2) 감정은 원핫 인코딩을 통해 레이블링을 진행
3) 인풋 데이터는 본 모델에서는 워드 인덱스를 사용하여 고유한 인덱스를 순서로 하는 벡터를 활용
4) keras의 Sequential 와 model_name.add 함수를 통해 LSTM, GRU를 자유롭게 쌓고, 파라미터로 레이어 갯수를 준비한 후, input_shape에 준비한 벡터 사이즈를 입력
5) model_name.fit 함수를 통해 준비한 훈련 인풋, 아웃풋, 검증의 인풋 아웃풋을 입력 후, 배치 사이즈, 반복 수를 입력
6) 학습 진행
7) model_name.predict 함수를 통해 새로운 인풋을 넣으면 그에 맞는 결과 값이 도출됨
