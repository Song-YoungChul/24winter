GTM

============================================
ugtm 모듈의 gtm means, gtm modes를 활용함

본 모델은 퍼져 있는 데이터를 대상으로 시각화 하는 도구임
예시로 특허 데이터의 명칭과 요약을 tf-idf 기법으로 임베딩 후 이를 시각화 하고자 함

============================================

사용법

1) 위즈 도메인에서 특허 데이터를 다운받아 엑셀 데이터 형태로 투입
2) 데이터를 불러온 후 명칭과 요약을 대상으로 Lemmatization, Stemming 진행 후 필요한 데이터 외 다 지우는 전처리 진행
3) 이후 사이킷런에서 지원하는 tf-idf 벡터화 함수 TfidfVectorizer를 이용하여 임베딩 진행
4) GTM 결과로 확인하고 싶은 라벨을 지정
5) ugtm에서의 egtm 모듈을 활용해 gtm means와 gtm modes를 구분하여 사용 가능함
	- means는 저차원 공간에서 각 노드들을 매핑한 것으로 클러스터링 또는 패턴 확인에 유용함
	- modes는 데이터의 분포를 시각화하고 이해에 도움을 줌
6) eGTM 함수를 통해 gtm 맵을 제작하며, 만들고 싶은 차원을 m에, model에는 "means" 또는 "modes"를 넣은 뒤, fit_transform함수에 제작하고 싶은 맵 데이터를 넣으면 완료됨
7) alt 함수를 통해 각 노드에 대한 라벨을 입력하면 시각화를 완료해 줌
